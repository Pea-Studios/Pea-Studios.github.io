About Us

Built by developers,
for server owners.

Pea Studios is a small development studio focused on building high-quality Minecraft plugins. Founded in 2026, our mission is simple: create tools that are powerful yet easy to use — plugins that work out of the box with familiar commands, so switching from Towny, Lands, or other plugins is seamless.

We target small and mid-sized servers because they deserve premium-quality tools at accessible prices. Every plugin we build is designed with real server owners in mind.

Core Features (Free Edition)

🏗️

Land Claiming

Chunk-based claiming system with selection tool, auto-claim mode, and claim blocks. Base 10 chunks + 5 per member.

👥

Roles & Permissions

4 default roles + custom roles with 61 granular permission flags covering blocks, containers, combat, animals, workstations, and utilities.

💰

Economy

Full Vault integration with land treasury, per-chunk costs, teleport costs, and configurable economic cycles.

⚔️

War System

Kill-point wars with live scoreboard, progress bars, and land transfer on victory. Declare, fight, conquer.

🔮

Beacon Warps

Unique public warp system — step on beacons to browse warps. Paginated GUI with search, categories, and custom icons.

🗺️

Map Integration

Dynmap & BlueMap support for visualizing claimed lands on your web map.

🔄

Migration

Built-in Towny migration support. Switch from Towny or Lands without losing data.

🛡️

Full Protection

Explosions, fire spread, mob spawning, PvP, pistons, liquid flow — all configurable per-land and for wilderness.

💬

Land Chat

Private communication channel for land members. Keep town discussions in-game.

PeaForestry

Timber-style tree felling plugin. Cut a tree at any point and watch it fall — realistic physics, directional falling, and stump decay.

TBA
Coming soon

Realistic Tree Felling

🪓

Cut Anywhere

Chop at the bottom and the whole tree falls. Cut in the middle and only the top half falls — the stump stays standing. Realistic split-point physics.

🌲

Directional Falling

Trees fall in the direction the player is facing. Logs rotate horizontal, leaves partially retained on sky-facing sides for a natural look.

💀

Stump Decay

The ground around the base decays after felling — dirt and grass degrade, leaving a natural clearing where the tree once stood.

🌳

All Tree Types

Oak, Birch, Spruce, Jungle, Acacia, Dark Oak, Mangrove, Cherry — all supported.

⚡

Animated Falling

One log per tick animation. Trees visually topple over rather than just disappearing.

🎛️

Configurable

Fall direction, drop items, leaf retention %, stump decay radius, drop delay — all adjustable in config.

🔧

Single-Tree Only

Only the tree you cut is affected. No chain reactions, no area damage. Clean, predictable behavior.

Pealo

Dynamic ELO-based PvP ranking system. Track combat performance, climb ranks, and dominate the leaderboard.

TBA
Coming soon

10-Tier Rank System

Copper 1000+ Bronze 1200+ Iron 1400+ Steel 1600+ Gold 1800+ Platinum 2000+ Diamond 2300+ Master 2600+ Grandmaster 3000+ Apex 3500+
📈

Dynamic ELO

Gain more ELO from beating stronger opponents. Base gain of 15 scales by opponent-to-player ELO ratio. Range: 1–40 per kill.

🪙

Kill Points Economy

Earn Kill Points (KP) from combat, spend them in the cosmetic shop. Dynamic scaling like ELO, range 1–50 KP per kill.

🛡️

Anti-Farm System

Prevents reward farming — kills on the same player stop rewarding after threshold (default 2). 180s window, 240s cooldown.

🤝

Assist Tracking

Deal 20%+ of total damage to qualify for assists. Earn 0.5x ELO and KP from assisted kills.

🏆

Leaderboard GUI

Top 100 players with pagination. Podium display for top 3, click-to-view stats. 60-second cache for performance.

🛒

Cosmetic Shop

Admin-editable shop GUI. Buy items or execute commands with Kill Points. Paginated with display previews.

Developer API

Build on top of our plugins.

Extend our plugins with addons, integrate with PlaceholderAPI, and hook into our events.

PeaTowny API Addon System

PeaTowny has a full addon system. Drop JAR files into plugins/PeaTowny/addons/ and they load automatically with their own lifecycle.

// Example: Accessing the PeaTowny API PeaTownyAPI api = PeaTownyAPI.getInstance(); // Land queries Land land = api.getLandAt(player.getLocation()); List<Land> lands = api.getLandsByOwner(playerId); // Permission checks boolean canBreak = api.hasPermission(landId, playerId, RoleFlag.BLOCK_BREAK); // Economy api.deposit(landId, 500.0); double balance = api.getBalance(landId); // Nations & Wars Nation nation = api.getNation(nationId); List<War> wars = api.getActiveWars();
